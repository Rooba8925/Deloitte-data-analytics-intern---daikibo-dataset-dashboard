# Deloitte Data Analytics – Daikibo Dashboard

## 📊 Project Overview

This project is a **data analytics dashboard** created using **Tableau** to analyze operational telemetry data from **Daikibo factories**.
The dashboard helps visualize machine performance, factory operations, and identify potential issues using interactive data visualizations.

The goal of this project is to demonstrate **data analysis, visualization, and business insights generation** using real-world industrial data.


## 🛠 Tools & Technologies

* Tableau Desktop / Tableau Public
* JSON Dataset
* Data Visualization
* Business Analytics


## 📂 Project Files

| File                                            | Description                                                  |
| ----------------------------------------------- | ------------------------------------------------------------ |
| `Deloitte data analytics daikibo dashboard.twb` | Tableau workbook containing the dashboard and visualizations |
| `daikibo-telemetry-data.json`                   | Dataset containing telemetry data from factory machines      |


## 📈 Dashboard Features

The dashboard includes visualizations such as:

* Machine status analysis
* Factory performance monitoring
* Telemetry data insights
* Operational issue detection
* Interactive filtering and exploration

These insights help understand **equipment performance and operational efficiency**.



## 📊 Dataset Information

The dataset used in this project contains **telemetry data from Daikibo industrial machines**, including information such as:

* Machine ID
* Factory location
* Operational status
* Sensor readings
* Machine performance metrics

## 🚀 How to Run the Project

1. Install **Tableau Desktop** or **Tableau Public**.

2. Open the file:

   
   Deloitte data analytics daikibo dashboard.twb
   

3. If the dataset path is missing, reconnect it:

   * Go to **Data Source**
   * Select `daikibo-telemetry-data.json`
   * Browse and reconnect the dataset.
